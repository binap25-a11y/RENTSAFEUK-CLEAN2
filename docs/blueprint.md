# **App Name**: Maintain UK

## Core Features:

- Welcome/Login Screen: User authentication via email/password and social media login.
- Dashboard: Display interactive cards with key metrics such as total properties, open maintenance requests, upcoming inspections, and quick actions.
- Add Property Form: Form to input property address, type, status (vacant, occupied, under maintenance), number of bedrooms/bathrooms, and tenant details with tenancy dates and a notes section.
- Property Image Upload: Ability to upload an image for each property.
- Maintenance Log: Log maintenance issues with descriptions and urgency levels.
- Inspection Scheduling: Schedule inspections with date/time pickers.
- Document Upload: Upload and store relevant documents like tenancy agreements and property certificates.

## Style Guidelines:

- Primary color: Soft blue (#A7D1AB), evoking trust and reliability.
- Background color: Very light desaturated blue (#F4F8F6) to ensure a clean and professional feel.
- Accent color: A muted green (#77A6B3) for calls to action.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet accessible feel.
- Card-based design for dashboard elements, promoting easy access and clear data presentation.
- Use of line icons to represent categories such as property type, maintenance, inspections, and document management, which creates a visually consistent design.
- Subtle animations during form transitions and loading states to enhance user experience.
